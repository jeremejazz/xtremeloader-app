# xtremeLoader App

Cordova project for XtremeLoader app. This is dependent to the xtremeloader project where the `build` files are deployed to the `www` folder.

As of now this is currently under development and not yet available on Google Playstore. But you can still build this to test in mobile device

#### Platform(s):
- Android 4.0+


#### Requirements
- Cordova 6.4
- Android SDK
